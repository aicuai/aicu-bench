Key Findings Summary

1. VRAM Zombie Process (R1 Fix)

R1 showed D: 186 tok/s vs others 19-21 tok/s. Root cause: zombie Ollama process consuming all VRAM, forcing CPU fallback. After cleanup, all drives ~189 tok/s.

2. Model Size Determines Storage Impact

Small (<7GB): No storage difference
Medium (~15GB): Up to 7.5x cold start difference (D: vs F:)
After VRAM cache: all drives identical

3. ICY DOCK Performance

Same Gen5 NVMe via ICY DOCK removable bay: ~30-40% penalty for 15GB+ model loads, negligible for <7GB models. Practical for swappable storage.

4. Parallel AI Workloads

98GB VRAM enables true multi-task AI: TTS + voice dialogue + image gen simultaneously with <25% VRAM usage, stable 65°C. Only video gen (>30GB) conflicts.
