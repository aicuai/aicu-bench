Het bevat selectors, eigenschappen en mappen. **Voorbeeld:** ```css body { font-family: Arial, sans-serif; background-color: #f0f0f0; } button { background-color: #4CAF50; color: white; padding: 10px 20px; border: none; cursor: pointer; } ``` --- ### 4. **Samenvatting van de punten** - **HTML** bevat de structuur van de webpagina. - **JavaScript** maakt de pagina interaktief. - **CSS** bepaalt de stijl en layout van de HTML. --- ### 5. **Voorbeeld van een volledig webpagina** ```html Geen Titel
